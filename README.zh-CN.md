# layaair-skills

[English](./README.md)

面向 AI Agent 的 LayaAir 引擎相关通用技能库。

[LayaAir](https://github.com/layabox/LayaAir) 是一款高性能、跨平台的游戏与互动内容引擎，支持 2D、3D、UI、动画、物理、资源管理、编辑器扩展和多端发布等能力。LayaAir 3D 引擎提供场景、相机、灯光、材质、网格、骨骼动画、粒子、物理、后处理等完整 3D 开发能力，适用于网页、小游戏、App、桌面端等多平台互动项目。

本仓库收集和维护与 LayaAir 开发相关的 AI Agent 技能说明，帮助 Agent 更准确地理解 LayaAir 工作流、命令行工具和 IDE 插件开发规范。

## 当前技能

| 技能 | 说明 |
| --- | --- |
| `layaair-cli` | LayaAir 命令行工具技能。用于创建项目、构建项目、校验资源文件、启动内置预览服务、运行项目中注册的脚本方法，以及管理 LayaAir CLI 版本。 |
| `layaair-ide-plugin` | LayaAir IDE 插件开发技能。用于编写编辑器插件、编辑器面板、菜单、检查器字段、构建插件、资源处理器、场景钩子、Gizmo 等 IDE 扩展能力。 |
